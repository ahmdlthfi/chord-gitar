Website Chord Gitar

Website ini merupakan project berbasis HTML yang menampilkan kumpulan chord lagu gitar dalam bentuk website statis. Pengguna dapat memilih lagu dan melihat chord gitar secara langsung melalui browser tanpa memerlukan login atau backend.

Website dapat diakses melalui GitHub Pages pada alamat berikut

Struktur File Project

index.html sebagai halaman utama website

semua-lagu.html sebagai halaman daftar lagu

kontak.html sebagai halaman kontak

song1.html sampai song10.html sebagai halaman chord lagu

bg.jpg dan songbg.jpg sebagai gambar latar website

Teknologi yang Digunakan

HTML5

CSS sederhana

Web browser seperti Google Chrome atau Mozilla Firefox

Tujuan Pembuatan Project

Menyediakan media pembelajaran chord gitar

Melatih pembuatan website statis menggunakan HTML

Memenuhi tugas perkuliahan pengembangan web dasar

Cara Menjalankan Project

Download atau clone repository

Buka folder project

Jalankan file index.html menggunakan browser

Website akan tampil dan dapat digunakan

Catatan

Website dihosting menggunakan GitHub Pages

Seluruh file source tersedia di repository GitHub

Project bersifat edukatif dan non komersial
